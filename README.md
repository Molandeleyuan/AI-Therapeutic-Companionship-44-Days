## 🧠 Research Core
This study presents a 44-day longitudinal single-case research, demonstrating the process of AI therapeutic companionship through collaborative song creation, with 48 original works co-created.
- **5-Layer Protocol Framework**: Temporal Structure, Emotional Ratio, Stage Evolution, Meaning Framework, Contextual Geology
- **3 Key Effects**: Rereading Effect, Frame-Guiding Effect, Terminology Clarification Effect
- **Authorization Trigger Mechanism**: Dual-authorization requirement for AI memory exposure
- **Explicitization Mechanism**: A user-initiated strategy to stabilize cross-instance memory retrieval
